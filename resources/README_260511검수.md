# 🎨 resources — 자산 풀 (Asset Pool)

> 모든 PNG·webp·sound 등 리소스의 단일 보관소 + 노출 보호 분리.

---

## 0. 위치

```
v1_260509/
├─ packages/         ← 코드 모듈
├─ shared-data/      ← 공유 스태틱 메타
└─ resources/        ← (여기) 자산 풀
```

---

## 1. 카테고리·노출 분리

```
resources/
├─ aidong/
│  ├─ public/         ← 누구나 fetch 가능 (board_icon, 도감 회색 실루엣 등)
│  └─ protected/      ← 권한 검증 후만 (body·face·signature)
├─ cutscenes/
│  ├─ public/         ← 누구나 (오프닝·튜토리얼)
│  └─ protected/      ← 영입·해금 후 (캐릭터 컷신)
├─ backgrounds/
│  ├─ public/         ← 마이섬·항구·보드
│  └─ protected/      ← 미해금 구역
└─ audio/
   ├─ public/         ← BGM 공통·UI SFX
   └─ protected/      ← 캐릭터 시그니처·해금 BGM
```

---

## 2. URL 컨벤션 (asset-helper.ts)

| 노출 | URL prefix | 동작 |
|---|---|---|
| public | `/assets/{category}/{path}` | Vite public 또는 정적 호스팅 직배포 |
| protected | `/api/resources/{category}/{path}` | backend 권한 검증 + stream |

**사용 예**:
```ts
import { publicAsset, protectedAsset } from '@idongworld/core'
publicAsset('aidong', '황금멍/board_icon.png')
  // → '/assets/aidong/황금멍/board_icon.png'
protectedAsset('aidong', '황금멍/body.png')
  // → '/api/resources/aidong/황금멍/body.png'
```

→ **직접 문자열 결합 금지** · 항상 헬퍼 사용.

## 4. 파일명 규칙

- 한글 캐릭터 ID 폴더 OK (예: `황금멍/`) — fs.allow 셋업 됨
- 파일명: `{partType}.png` 또는 `{partType}_{state}.png`
  - body.png / face_default.png / face_smile.png / signature.png / board_icon.png
- 해상도: 1024×1024 표준 (캐릭터) · 가변 (배경)
