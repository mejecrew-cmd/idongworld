# 🧩 M02. 오프닝 모듈 — 상세 명세 (샘플)

> **이 문서가 "모듈 상세"의 템플릿**이다. 화면별로 `영역·목적·진입·이탈·UI·대사 샘플·필요 데이터·상태·미결`을 채운다.
> 이후 M03~M20도 이 틀을 따른다.

## 0. 개요
- **위상**: 1회성·단방향. *플레이어 → 플레이어캐릭터 다이브* + *첫 아이동 영입* + *회색섬 → 하트섬 전환*.
- **헌법 정합**: "첫 3분 안에 본 게임 시작" + 회복의 여정(의도문서 §4).

## 2. 화면별 상세 명세 ★ (← 샘플 포맷)

### SPLASH-00 · 스플래시/계정 진입 *(M01 경계)*
- **영역**: 앱유저(L0)
- **목적**: 앱 시작·자동로그인 체크. 게임 바깥(로그인/약관)과 분리.
- **진입**: 앱/웹 최초 로드
- **이탈→다음**: `게스트로 시작` → PLAYER-00 / 자동로그인 토큰 有 → 분기
- **UI**: 로고·버전·로딩 / 버튼 `[게스트로 시작]`
- **데이터**: `firstRun`, `authToken?`
- **상태**: ✅ (게스트=오프닝 연계, M01)

### PLAYER-00 · 타이틀
- **영역**: 플레이어(L1)
- **목적**: 입장 + 최초실행 분기.
- **진입**: SPLASH 통과
- **이탈→다음**: `입장` + 최초실행 → OP-01 / 기존 저장 → ISLAND-HUB
- **UI**: 버튼 `[입장]` `[오프닝 스킵]` `[처음부터 다시 보기]`
- **규칙**: `오프닝 스킵` → LAND-04(이름짓기)로 직행 → 저장 후 허브
- **데이터**: `firstRun`, `hasSave`
- **상태**: 🟢

### OP-01 · 현실 방
- **영역**: 플레이어(L1) → PC 다이브 직전
- **목적**: 일상의 윤슬 인식. "환상의 결" 시작점.
- **연출 샘플**: 새벽 방 · 불투명 창에 맺힌 가로등 빛이 윤슬처럼 · 최애 포스터. *인물은 상징만(얼굴/성별 X)*.
- **대사 샘플**: (독백, 음성 X) "…저 반짝임, 윤슬 같다."
- **이탈→다음**: 버튼 `[내일을 위해 숙면에 든다]` → OP-02 / `[오프닝 스킵]`
- **데이터**: —
- **상태**: 🟢

### OP-02 · 바다와 조각배
- **영역**: 플레이어캐릭터(PC-선장) 진입
- **목적**: 윤슬 은유 → 진짜 바다. 플레이어 = 조각배 선장 자각. 멀리 회색섬.
- **연출 샘플**: 침대 흔들림 → 바다 윤슬 · 어푸어푸 소리.
- **선택지 (분기)**: `[어푸어푸를 살펴본다]` / `[큰 섬으로 전속력 돌진!]`
- **분기 규칙**: 둘 다 OP-03으로 수렴. *돌진* 선택 시 "아이동이 헤엄쳐 쫓아오는" 추가 텍스트.
- **데이터**: `openingBranch: 'look' | 'dash'`
- **상태**: 🟢

### OP-03 · 어푸어푸 구조
- **영역**: PC-선장
- **목적**: 첫 아이동 등장·구조 = 첫 만남 성립.
- **대사 샘플**: 🟠[첫 아이동]: "어? 어푸어푸! 살려주세요!" … (길고 실없는 수다)
- **이탈→다음**: 버튼 `[젖은 병아리 같은 존재를 본다]` → OP-04
- **데이터**: `recruitedFirstAidong = true`
- **상태**: 🟢 / 🟠 **첫 아이동 정체 미결**(§4)

### OP-04 · 아이동 소개·상륙
- **영역**: PC-선장 → PC-섬주인 전환
- **목적**: 아이동 = "자기 섬 가진 존재" 소개. 큰 섬 솟음 = "섬주인 방문". 아이동 = 임시 안내자.
- **대사 샘플**: 🟠[첫 아이동]: "어 너는 배도 있더니 섬도 있어? …좋아, 결정했어. 너! **나를 키워라.**" → *("키운다"는 친구의 어휘. 시스템상 호스트는 키우지 않음 — 다마고치 코어 정합)*
- **이탈→다음**: 버튼 `[상륙]` → LAND-01
- **데이터**: —
- **상태**: 🟢

### LAND-01 · 마이섬 첫 상륙 지도
- **영역**: 마이섬(L3) · PC-섬주인
- **목적**: 15구역 전체지도 첫 노출.
- **연출 샘플**: 우하단 항구=현위치(활성) · 중앙 숙소부지=활성(폐허) · 나머지 13구역=회색·구름(잠금).
- **이탈→다음**: 버튼 `[숙소 방문]` → LAND-02
- **데이터**: `island.zones[].state`
- **상태**: 🟢

### LAND-02 · 숙소 재건 전
- **영역**: PC-섬주인 · 숙소 모듈 예고
- **목적**: 폐허 숙소 = "기본 생활 기반 없음" 문제 제시. 아이동이 청소 맡고 섬 둘러보라 안내.
- **대사 샘플**: 🟠[첫 아이동]: "여기가 내가 살 곳이야? 근데… 자기 방은 자기가 치워야지! 넌 네 섬을 둘러봐 줘!"
- **이탈→다음**: 버튼 `[퀘스트 확인]` → LAND-03
- **데이터**: `lodgeBuilt = false`
- **상태**: 🟢

### LAND-03 · 14구역 둘러보기 (숙소 제외)
- **영역**: 마이섬(L3)
- **목적**: 숙소 제외 14구역 라벨 클릭 → 이름·이미지·설명 확인 → 진행률 체크.
- **규칙**: 항구 팝업 = "내 조각배 정박 / 지금은 떠나지 않음" · 그 외 구역 = `[지금은 입장 못함]`. **14개 확인 완료 → 숙소 라벨 활성화**.
- **이탈→다음**: 숙소 라벨(활성) → LAND-04
- **데이터**: `visitedZones[]`
- **상태**: 🟢

### LAND-04 · 숙소·섬 이름짓기
- **영역**: PC-숙소주인
- **목적**: 숙소 = 방 2개 오두막으로 재건. 숙소명 → 섬명 입력·저장.
- **UI**: 텍스트 input 2단 (숙소명 → 섬명). placeholder "예: 별빛 숙소…"
- **연출 샘플**: 회색→풀컬러 변환 + "아이동 월드 OO섬 라이프 시작!" 완료 팝업.
- **대사 샘플**: 🟠[첫 아이동]: "근데… 저 방은 뭐야? 친구가 있어야지! 우리, 이름 짓자!"
- **이탈→다음**: 저장 → ISLAND-HUB
- **데이터**: `lodgeName`, `islandName`, `lodgeBuilt = true`
- **상태**: 🟢

### → ISLAND-HUB *(M03으로 인계)*
- 오프닝 종료. 첫 도감 등재(첫 아이동·하트섬) + 항해 잠금해제.

## 4. 미결 · 쟁점 · 고려사항 (M02)

### 4-1. 미결 (결정 필요)
| ID | 안건 | 선택지 |
|---|---|---|
| ✅O-1 🔒 | **첫 아이동 = 고정·병아리형** (본진 후보 중 병아리상 1명) | 확정(6/9). 최종 인물 = 본진 확정 시 |
| O-2 | **4시간 통과의례 길이** | 4시간 유지 / 압축(즉시) / 단축 옵션 |
| O-3 | 게스트 → 소셜 로그인 **바인딩 시점** | 오프닝 종료 직후 / 첫 저장 / 임의 |
| O-4 | 컬러 복구 연출 | 4단계 점진 / 1회 변환(5초) |
| ✅O-5 🔒 | 항해 이동(LAND 이후) | 지정 이동 기본 + 주사위 옵션(M05 확정) |

### 4-2. 쟁점 (대립·트레이드오프)
| 쟁점 | A 입장 | B 입장 | 메모 |
|---|---|---|---|
| 첫 아이동 | **별꼬리 고정**(서사·엔진 d락 #1=혜성) | **본진 후보서 선발**(가챠) | 오늘 본진 재구성(VANNER+ATEEZ+SKZ)과 충돌. 별꼬리=혜성 매핑 흔들림 |
| 4시간 의식 | **리텐션 장치**(1h마다 재방문 4회) | **진입장벽**(6/5 "일반인 너무 복잡") | 헌법 "첫 3분 본게임"과도 긴장 |
| 항해 이동 | **PoC 구현·항해 맛** | **첫 사용자 난이도**(6/5 반론) | ✅ M05에서 지정 이동 기본 + 주사위 옵션 확정 |
| 게스트 진행 | **마찰↓ 즉시 플레이** | **바인딩 실패 시 데이터 유실** | 로컬→서버 이관 안전장치 필요 |

### 4-3. 고려사항 (설계 시 유의)
- 이름 입력: **욕설 필터·길이 제한**
- 복귀 알림 강도(헌법 §광고·푸시 강제 X)
- 통과의례 **1회 한정 vs 캐릭터 추가 시 짧은 변형 의식**
- 첫 4시간 보상의 **경제 균형**(Phase 1 미니게임 영향)
- 게스트 진행 데이터 = **로컬 저장 → 바인딩 시 서버 이관**
- 오프닝 스킵을 **게스트도 허용**할지
- IP 안전: 첫 아이동이 실존 오마주면 **노출 룰** 적용
