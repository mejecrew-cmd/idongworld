# 🧩 M15. Visual View·Fandom Battle — 상세 명세

> **위상**: 캐릭터 키워드를 Node-Edge 그래프로 시각화하는 팬덤 트렌드 지형도 + 하트 투표 경쟁.
> 🟢 · ✅ · 🟡 · 🟠

## 0. 개요
- 전 세계 유저가 실시간으로 만드는 팬덤 트렌드. 1위 키워드 → 해당 캐릭터 **이번 달 대사 테마**(M10 연계).

## 1. 기획 대조
- GDD §16(키워드 그래프·하트 투표·월간 결산) / AI 페르소나(M10) 키워드 입력 → ✅ 통합.

## 2. 화면별 상세
- **VVIEW**: 키워드 Node-Edge 그래프 · 키워드 등록 · 하트 투표 · My Binder/Marketplace 이중 뷰.

## 3. 서브모듈 · 예시값 (불렛)
- **참여 흐름**: 키워드 등록(연상 단어 자유 입력) → 하트 투표 → 월간 결산
- **투표 규칙**: 1일 1계정 10하트 · 1키워드 최대 3하트 · 자기 키워드 투표 불가 · 하트 증폭기 ×3
- **월간 결산**: 1위 → 이번 달 대사 테마 / 2~5위 → 꿈 일기 소재 / 최다 유지 → 전용 칭호·한정 포토카드
- **이중 뷰**: My Binder(내 키워드·투표) · Marketplace(전체 랭킹·실시간 트렌드)

## 4. 미결 · 쟁점 · 고려
- 미결: 어뷰징(멀티세션 6/5) 방지 강도
- 쟁점: 개방형 키워드 ↔ 금칙어/실명/IP 노출 → 서버 검증 + 금칙어 필터(M10·M14 공통)
- 고려: 1계정 일일 한도·자기 투표 불가로 어뷰징 차단 · Fandom Battle 결산(M18)과 연동
