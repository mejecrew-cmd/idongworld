# 🧩 M08. 성장·데뷔 — 상세 명세

> **위상**: 숙련도 훈련(숙소) → 진화 → **무대(구역)에서 데뷔**. 무대 = 구역(M06)의 한 종류, 핵심 가치 아닌 한 구역 콘텐츠(위상 격하 유지).
> **상태**: ✅ 확정 · 🟡 기획중 · 🟠 미결

---

## 0. 개요
- **숙련도**(숙소 연습실/GYM·M04) → **진화 4단계** → **데뷔 스테이지(무대 구역·M06)** → 포토카드(M11).
- 도감 아이템(아이동 귀속) = **진화 연료**(6/8).

---

## 1. 기획 대조 결과 ★ (데뷔 점수 모델 충돌)

| 항목 | S02 데뷔(5/11) | GDD §14 | 채택 |
|---|---|---|---|
| **데뷔 점수** | **5 태스크 별점**(연습·포즈·리허설·메이크업·무대, 합 5~17) | **숙련도 점수**(보컬×10+댄스×10+비주얼×5)+보정 | ✅ 숙련도 기반 + 5태스크 연출 |
| 등급 | C/B/A/S (별점) | S/A/B/C (90/70/50pt) | 🟠 임계 통일 |
| 해금 | 친밀도 Lv5 + 시그니처 | 보컬 Lv1+ | 🟠 |
| 보상 | 포토카드(등급별)·코인·스테이지코인 | 포토카드+누적보상+SUNO | ✅ 통합 |
| 진화 | 친밀도 Lv 외형(다마고치) | **4단계**(연습생→데뷔조→활동→탑) | ✅ 4단계 |
| SUNO AI | — | BGM 생성·SNS 공유 | ✅ 추가 |
| 무대 위치 | 마이섬 한 구역(위상 격하) | 별도 | ✅ **구역(M06)** |

> **핵심**: 데뷔 점수가 **5 태스크 별점(KKH식)** 인가 **숙련도 누적 점수(GDD)** 인가. 전자=즉석 미니액션, 후자=장기 육성 반영. 절충: 숙련도 기반 + 태스크 연출.

## 2. 화면별 상세 명세

### TRAIN · 연습실/GYM (숙소 내)
- **목적**: 숙련도 4종 훈련(보컬·댄스·비주얼·글로벌).
- **위치**: 숙소(M04) 실내 편의시설
- **상태**: 🟡

### UPGRADE · 진화
- **목적**: 4단계 진화 — 외형·콘텐츠 해금.
- **단계**: 연습생 → 데뷔조(친밀도Lv10+댄스Lv5+스테이지3회) → 활동(친밀도Lv25+전숙련Lv7+AI180일) → 탑(친밀도Lv30+전구역완성)
- **연료**: 숙련도 + **도감 아이템(아이동 귀속·25종, 6/8)** + 연출(별빛 파티클)
- **상태**: ✅

### DEBUT · 데뷔 스테이지 (무대 구역)
- **위치**: **무대 = 구역(M06)** — 구역마다 가능, Phase1 대표 1무대
- **흐름**: 해금 → 숙련도 점수 산정 → 5 태스크(연습/포즈/리허설/메이크업/무대) 연출 → 등급 → 포토카드(M11)
- **점수**: ✅ G-1 (숙련도 기반 + 5태스크 연출)
- **등급**: S/A/B/C → 포토카드 등급 확정·코인·누적 보상
- **SUNO AI**: 키워드 → BGM 생성 → 스테이지 영상 SNS 공유
- **보안**: 클라 별점 신뢰 X, 서버 재계산
- **상태**: 🟡

## 3. 숙련도·진화·점수

| 숙련도 4 | 보컬·댄스·비주얼·글로벌 (Lv10), 액티브 스킬 6 |
| 진화 4단계 | 연습생→데뷔조→활동→탑 (외형·칭호·해금) |
| 데뷔 점수(GDD) | (보컬×10+댄스×10+비주얼×5) + 파라미터·장비 보정 → S≥90/A70/B50/C |
| 누적 보상 | 3회=데뷔의상조각 / 5회=데뷔조진화 / 10회=스킬 / 30회=명예의전당 / 50회=전설카드 |
