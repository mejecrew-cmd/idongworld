# 🧩 M03. 마이섬 허브 = 메인 맵 — 상세 명세

> **정체**: 마이섬 허브는 **고정 15구역(슬롯)을 지도 형태로 제공하는 사실상의 메인 화면(메인 맵)**.
> 지도에서 **구역(슬롯)을 누르면 해당 모듈로 진입**한다. (전체 맵 → 모듈)
> **상태**: 🟢 구현(PoC v3) · ✅ 확정 · 🟡 기획중 · 🟠 미결

---

## 0. 개요
- **정체**: 마이섬 **메인 맵**. 앵커 2(항구·숙소) + 편입 슬롯 13 = 고정 15구역을 지도로 표시.
- **역할**: 매 접속의 시작점. 구역 탭 → 모듈 진입. = "전체 맵에서 눌러 들어가는" 허브.
- **구조**: `메인 맵(전체) → 구역 탭 → 각 모듈(모듈맵/내부)`.

## 2. 화면별 상세 명세

### MAP-MAIN · 마이섬 메인 맵 (허브)
- **영역**: 마이섬(L3) — 본 게임 메인 화면
- **목적**: 고정 15구역을 **지도로 제공** + 구역 탭으로 모듈 진입.
- **진입**: 오프닝 종료 / 타이틀(기존 저장) / 하단탭 `홈` / 모듈에서 뒤로
- **UI 요소**
  - **지도**: 15구역 다이오라마 (앵커 2: 항구·숙소 + 편입 슬롯 13)
  - **구역 상태**: 잠금(회색·구름) / 편입됨(아이동 점유) / 활성
  - **오버레이 HUD**: Fandom Level · 코인 · 별빛 · 설정 · 다국어 · `책임감 있는 결제`
  - **하단 탭(5)**: 홈 · 육성 · 커뮤니티 · 콘텐츠 · UGC마켓
  - **Quick Access(재정의)**: 친구 · 마스코트 정보 · 창고 · 상점 · 이벤트·우편 · 설정
  - **풀샷 모드**: 줌아웃 15구역+아이동 한 화면 → SNS 공유(html2canvas)
- **이탈→다음 (구역 탭)**
  | 탭한 구역 | 진입 모듈 |
  |---|---|
  | 항구(앵커) | M05 항해 |
  | 숙소(앵커) | M04 숙소 |
  | 편입 슬롯(아이동 구역) | M06 구역(미니게임·배치) |
  | 아이동 정보 | M09 도감 |
- **데이터**: `island.zones[].{themeFixed, occupant, state}` · currency · fandomLevel
- **상태**: 🟢 (PoC LAND-01 전체지도 기반) / 🟡 (풀 UI)

### (참고) 캐릭터 케어 대시보드 — ✅ 숙소 내부 확정 (6/8 유저)
- 말풍선 대화·파라미터·케어 액션 = **숙소 내부(M04)** 에서 수행. (메인맵 = 지도이지 케어 화면 아님)
- **무대 = 구역**(메인맵의 한 구역, 또는 구역마다) — M06·M08. 숙소·홈이 아님.
