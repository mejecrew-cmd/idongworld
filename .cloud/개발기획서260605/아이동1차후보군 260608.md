# 아이동 1차 후보군 (260608)

> **목적**: 본진(런칭 핵심 아이동) 확정에 앞서, 어떤 K-POP 그룹을 영감원으로 삼아 아이동 캐릭터를 구성할지
> **1차 후보군과 선정 의도·기준·방식**을 고정한다. 이 문서는 본진 12인 최종 큐레이션과 페르소나 엔진 d·e 작업의 입력 자료다.
> 결정 등급: **후보군 = 방향 확정 / 본진 12인 = 미확정(다음 단계)**.

---

## 1. 선정 의도 (Why)

1. **게임 정체성과의 정합** — 아이동월드의 코어 루프는 *항해·항구·배·재기 서사*다. 후보군은 이 세계관과 톤이 맞아야 한다.
2. **오마주 방식** — 실존 그룹을 **페르소나 기반(영감)** 으로만 차용한다. 따라서 멤버의 현재 활동상태(탈퇴·군입대·해체)는 **선정 기준이 아니다.** 단, 페르소나 엔진에 먹일 **공개 자료(성격·서사·비주얼)가 충분한가**(= 아이동화 가능성)는 핵심 기준이다.
3. **재기(언더독) 서사 앵커 + 팬덤 화력의 결합** — 피크타임 우승팀 VANNER를 서사 앵커로 두되, 글로벌 팬덤 그룹을 더해 캐주얼·팬 유입을 확보한다.
4. **동물 종(種) 편중 회피** — 전체 영감원 풀이 강아지·고양이로 심하게 쏠려 있어(아래 §2-3), 본진이 "강아지·고양이 게임"으로 보이지 않도록 종 다양성을 의도적으로 설계한다.

> ⚠️ **IP 안전 원칙**: 게임 내 실명·소속사·그룹명·직접 식별 표현 금지. 전부 동물 페르소나로 추상화. 실제 컬래버 계약 전에는 "○○ 공식 캐릭터" 노출 불가. (정체성 문서 원칙 계승)

## 2. 선정 기준·방식 (How)

### 2-1. 4대 선정 기준
| 기준 | 측정 |
|---|---|
| ① 인기·팬덤 규모 | 글로벌 메가 / 대형 / 니치 (질적 등급) |
| ② 아이동화 여부(자료) | 캐릭터DNA 엔진 `e_아이동캐릭터` 매핑 완료 수 = 공개 자료 충분도의 대리지표 |
| ③ 동물 분포(종수) | 그룹 멤버들이 매핑된 **서로 다른 동물 종 수** |
| ④ 동물 편중 해소 기여 | 희귀·부족 종(강아지·고양이 외)을 채워주는가 |

### 2-2. 캐릭터DNA 엔진 5단계 파이프라인
영감원 → 아이동 캐릭터는 다음 단계를 거친다(`피크타임_출신.md` §0):

`a 라인업 등록 → b 자료 크롤링 → c 페르소나 분석 → d 페르소나일련번호+동물 매핑 → e 피지컬 결합`

- 후보군 4그룹은 모두 **e 단계(아이동 캐릭터화)까지 완료**되어 즉시 활용 가능.
- **본진 확정 멤버만 d 단계 락(페르소나일련번호 부여)**: 현재 VANNER 혜성(#1)·성국(#2) 2명 락 완료.

### 2-3. 동물 편중의 실태 (전체 풀 기준)
엔진 전체(672 아이동)의 동물별 작업량은 강아지·고양이로 크게 쏠려 있다:

```
강아지 120 · 고양이 89 · 토끼 65 · 여우 47 · 곰돌이 36 · 햄스터 29 · 사슴 26 · 병아리 25 · 늑대 24 …
```

→ 따라서 본진 선정의 관건은 "유명 그룹을 고르는 것"이 아니라, **희귀·부드러운 종을 채워줄 그룹을 함께 고르는 것**이다.

### 2-4. 멀티 마스코트 큐레이션 (핵심 메커니즘)
**아이동 1명(멤버)이 동물 마스코트를 2개 이상 가진다.** (예: VANNER GON = 고양이+공룡 / ATEEZ 산 = 강아지·고양이·늑대·여우 4종)
→ 멤버마다 후보 동물이 여러 개이므로, **그룹을 잘 고르면 종별로 1명씩 골라 담아 편중 없는 본진을 큐레이션**할 수 있다. 도감템 수 ≠ 멤버 수인 이유.

## 3. 1차 후보군 (4그룹)

> 코어 3그룹(VANNER·ATEEZ·STRAY KIDS) + 선택 1그룹(TXT). 멤버 → 동물(아이동명) 매핑은 엔진 `e_아이동캐릭터` 실데이터.

### 3-1. VANNER (배너) ⚓ — 서사 앵커
- **인기/팬덤**: 니치 (JTBC 〈피크타임〉 2023 **올픽 우승팀**). 게임의 *재기 서사*와 직결.
- **아이동화**: 7~8 매핑. **본진 2명 이미 확정(d·e 락)**.
- **동물 분포**: **7종** (희귀종 친칠라·공룡·개구리 보유 → 편중 깨기 선발대)

| 멤버 | 동물 | 아이동명 | 비고 |
|---|---|---|---|
| 혜성 | 햄스터 | 별꼬리 | ★본진 확정 #1 (locked) |
| 성국 | 친칠라 | 모카잔 | ★본진 확정 #2 (locked) |
| 태환 | 강아지 | 온기마당멍 | |
| GON | 고양이 / 공룡 | 발도장냥 / 온기렉스 | 멀티 마스코트 |
| 영광 | 여우 / 개구리 | 리본온기여우 / 온기개굴 | 멀티 마스코트 |

### 3-2. ATEEZ (에이티즈) 🏴‍☠️ — 세계관 정합 + 글로벌 화력
- **인기/팬덤**: 글로벌 메가(ATINY). **해적·항해·보물 세계관 = 게임 정체성 그 자체** (대표가 콜라보 예시로 직접 언급).
- **아이동화**: 16 매핑.
- **동물 분포**: **10종** (카리스마 희귀종 늑대·퓨마·기린·뱀 보유)

| 멤버 | 동물(아이동명) |
|---|---|
| 홍중 | 퓨마(캡틴퓨마) |
| 성화 | 고양이(숨결촛불냥) / 뱀(숨결뱀) |
| 윤호 | 강아지(항해리트리버) |
| 여상 | 강아지(숨결등대멍) |
| 산 | 강아지(스윙멍) / 고양이(샤인냥) / 늑대(포근늑대) / 여우(소프트여우) |
| 민기 | 햄스터(코지햄찌) / 기린(별빛기린) / 병아리(브리즈병아리) |
| 우영 | 고양이(깜장지냥) / 곰돌이(숨결곰) |
| 종호 | 강아지(사과멍) / 곰돌이(사과곰) |

### 3-3. STRAY KIDS (스트레이 키즈) — 팬덤·다양성 최상위
- **인기/팬덤**: 글로벌 메가(STAY).
- **아이동화**: 21 매핑.
- **동물 분포**: **17종** (코알라·라마·양·돼지·친칠라 등 → **동물 편중 최강 해소**)

| 멤버 | 동물(아이동명) |
|---|---|
| 방찬 | 늑대(리더늑대) / 코알라(리더코알라) |
| 리노 | 고양이(비트냥) / 토끼(까칠토끼) |
| 창빈 | 고양이(코지냥) / 곰돌이(양털곰) / 양(브리즈양) / 토끼(포근토끼) / 돼지(돼끼비트) |
| 현진 | 라마(여유라마) / 오리(온기오리) |
| 한 | 다람쥐(샤인다람) / 친칠라(비트친칠라) / 햄스터(루미햄찌) / 쿼카(스파크쿼카) |
| 필릭스 | 고양이(저음냥이) / 병아리(햇살병아리) |
| 승민 | 강아지(청량멍) / 고양이(루미냥) |
| 아이엔 | 여우(막내여우) / 사막여우(막내사막여우) |

### 3-4. TXT (투모로우바이투게더) — 매핑 1위 + 부드러운 희귀종 (선택)
- **인기/팬덤**: 대형(MOA).
- **아이동화**: **24 매핑 (전 그룹 1위)**.
- **동물 분포**: **16종** (다람쥐·사슴·펭귄·돌고래 등 부드러운 희귀종 → ATEEZ 강한 톤 보완)

| 멤버 | 동물(아이동명) |
|---|---|
| 연준 | 고양이(런웨이냥) / 다람쥐(런웨이다람) / 여우(런웨이여우) / 사막여우(런웨이사막여우) / 오리(런웨이오리) / 토끼(런웨이토끼) |
| 수빈 | 강아지(노블멍) / 늑대(소프트늑대) / 다람쥐(스텝다람) / 토끼(소프트토끼) / 쿼카(콩알볼) |
| 범규 | 강아지(하모니멍) / 곰돌이(소프트곰) / 호랑이(꼬마호랑이) |
| 태현 | 고양이(또렷초점냥) / 다람쥐(또렷다람) / 여우(또렷초점여우) / 앵무새(또렷앵무) |
| 휴닝카이 | 강아지(글로우멍) / 다람쥐(새끼다람) / 사슴(스윙노루) / 펭귄(비트펭) / 돌고래(샤인돌고래) / 병아리(비트병아리) |

| 5 | 늑대 | 산 | ATEEZ | 희귀 |
| 6 | 곰돌이 | 종호 | ATEEZ | 기본 |
| 7 | 코알라 | 방찬 | STRAY KIDS | 희귀 |
| 8 | 다람쥐 | 한 | STRAY KIDS | 희귀 |
