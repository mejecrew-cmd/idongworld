## 1. 현재 결정사항 (4개 소스가 합의/구현한 확정분)

### 1-1. 정체성·세계관 (확정)
- **한 줄 정의**: "내 아이돌의 상징 동물을 직접 키우고, 꾸미고, 교감하는 글로벌 팬덤 육성 시뮬레이션" (GDD §1.1)
- **장르 4축**: 코어=육성시뮬 / 확장=생활시뮬 / 소셜=팬덤 커뮤니티(UGC·Fandom Battle·포토카드) / 기술=AI 페르소나
- **오마주 원칙**: 현실 아이돌의 **4계층(캐릭터 영역) 동물 페르소나** 오마주. **실명·초상·소속사 비노출**, 팬덤 공유 상징(동물·기믹·패션 코드)만 사용 → 법적 독립 오리지널 (GDD §2.3, 변경가이드 IP룰, 일관)
- **핵심 감정 4**: 교감 / 성취 / 소유(세상 단 하나의 포토카드) / 공유

### 1-2. 온보딩·진입 (PoC v3에서 **실제 구현 완료**)
- **영역 분리**: 앱유저(L0) → 플레이어(L1) → 플레이어캐릭터(L2: PC-선장/섬주인) → 마이섬 게임(L3) (UI사이트맵 v2 / PoC v3)
- **오프닝 4씬**: 현실 방 → 바다·조각배 → 어푸어푸 구조(첫 영입, 가챠 1/retry X) → 상륙
- **회색섬→하트섬 의식**: 첫 상륙 → 15구역 둘러보기 퀘스트 → 숙소 재건 → 숙소명·섬명 저장 → 허브
- **상태 구조**: `firstRun/openingBranch/visitedZones/lodgeBuilt/lodgeName/islandName` (React/Next.js 전환 전제로 분리됨)

### 1-3. 데일리 코어 (확정)
- **5대 케어 액티비티**: 기상 / 아침·점심·저녁 식사 / 재우기 (GDD §5)
- **생존 파라미터 4종**: Hunger·Clean·Mood·Energy (0~100 자연 감소), 미수행 패널티 6/12/24/48h
- **부가 케어**: 목욕·산책·음악·미니게임·그루밍·편지 / **음식** 5분류(간식·일반·특식·음료·시그니처)
- **UI 원칙**: 수치 직접 노출 금지, 표정·말풍선·자연어로 상태 전달

### 1-4. 성장 (확정)
- **성장 스탯**: 친밀도 Lv30 · 팬덤XP(무제한) · 보컬/댄스/비주얼/글로벌 숙련도 Lv10 · Stage Power

- **성장 진화 4단계**: 연습생 → 데뷔조 → 활동 아이돌 → 탑 아이돌 (진화 연출 + 진화 포토카드)
- **데뷔 스테이지**: 점수=(보컬×10+댄스×10+비주얼×5)+파라미터·장비 보정, S/A/B/C 등급, 누적 보상, **SUNO AI BGM 생성·SNS 공유**

## 4. ~~충돌 2~~ ✅ 해소(6/8) — 탐험·구역: **고정 15구역 슬롯 + 항해 편입**

세 소스가 "어떻게 새 아이동을 만나고, 구역이 무엇인가"에서 **근본적으로 다르다.**

| 모델 | 새 아이동 만남 | 구역의 정체성 | 이동 방식 | 출처 |
|---|---|---|---|---|
| **A. 항해 부루마블** | 항로 보드 → 아이동 섬 상륙 | 항구·항로·바다판 | 주사위/지정이동 | 기존 모듈 + 6/5 |
| **B. 가변 구역** | 만남마다 메인에 구역 1개 추가 | **아이동별** 구역(미니게임+스킨) | (항해로 만남) | 6/5 회의 결정 |
| **C. 지역 탐험(고정)** | 여행자 NPC → 서브 마스코트 입주 | **고정 15구역**(정서 풍경, 조건 해금) | Energy -20/회, 1일 5회 | GDD v0.3 + PoC v3 |

**문제**: 
- GDD/PoC는 **C(고정 15 정서 풍경)** 에 깊이 의존 — 8장 탐험·12장 친밀도 해금·2장 세계관·17장 마이룸 확장이 모두 15구역 전제.
- 6/5 회의는 **B(아이동별 가변 구역)** 로 전환 결정 + **A(항해)** 를 메타 이동으로 둠.
- PoC v3 "다음 판단 #4 = v4에서 항구·배·항로·바다 보드를 붙일지" → 6/9 이후 **POC v3 재정의 범위**로 승격 검토.

**✅ 확정안 (2026-06-08 대표)**:
> **고정 15구역(정서 풍경, 불변)을 슬롯화** → **항해 부루마블(A)** 로 만난 **아이동 섬을 슬롯에 편입(채움)**. 구역 수는 15로 고정(**무한 가변 아님**). 각 구역에서 미니게임·자원·도감 동작.
> 참고: 6/5 [48:19] "15구역 중 숙소·한 구역 빼고 **13개 지정 게임**" = 항구·숙소를 뺀 **13개 슬롯이 아이동 섬 편입 대상**.
> 세부 편입 방식은 6/9 M06/09에서 기본 규칙 LOCK. 수치·UX·해제 비용은 후속 밸런싱.

→ 잔여 미결: 수치·밸런싱, 본진 최종 락, 게스트 바인딩, age-gate 세부.
