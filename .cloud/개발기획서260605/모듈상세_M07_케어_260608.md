# 🧩 M07. 케어 (다마고치) — 상세 명세

> **위상**: 단독으로 게임이 성립하는 본진. **숙소(M04)에서 구동**. 친구가 욕구·시간 위에 살고, 호스트는 짧게 들러 응답.
> **상태**: ✅ 확정 · 🟡 기획중 · 🟠 미결

---

## 0. 개요
- **호스트 = 동거인**(매니저·보호자 아님). 메인 동사 = "산다"(키운다 X).
- **하루 표준 30접속** = 메인 5(시간대 케어) + 서브 10(우연 케어) + 자유 15.
- **메인 5번 = 5 시간대 일기 5조각과 1:1 매핑**(M10 연계).

---

## 1. 기획 대조 결과 ★ (모델 충돌)

| 항목 | 다마고치 코어(SoT) | GDD §5·6 | 채택 |
|---|---|---|---|
| **욕구/파라미터** | 6 욕구(0~10) | 4 파라미터(0~100) | ✅🔒 **4파라미터 + 밀림 로직**(6/9) |
| **감소 방식** | 시퀀스 -1·밀림 허용 | 시간 자연 감소 | ✅🔒 **시퀀스+밀림 허용**(6/9) |
| **친밀도** | Lv 0~10 | Lv 1~30 | ✅🔒 **Lv30**(6/9) |
| 5대 케어 | 깨우기·아침·점심·저녁·재우기 | 기상·아침·점심·저녁·재우기 | ✅ 동일 |

| 미수행 | **친밀도 안 깎음**, 일기 톤만 변함 | 6/12/24/48h 패널티 | 🟠 강도 |
| 진화 | 친밀도 Lv 마일스톤 외형·액션 | 4단계(M08) | ✅ M08 |

> **핵심 결정**: 파라미터 모델이 **6욕구(Lv10·시퀀스·밀림)** 인가 **4파라미터(Lv30·시간감소)** 인가. 다마고치 코어가 "안 들러도 손해 X" 헌법에 더 정합. 6/5 단순화 기조 ↔ 다마고치 정교함.

## 2. 화면별 상세 명세

### CARE · 케어 (숙소 내 모달/씬)
- **영역**: 숙소(M04) 내부
- **목적**: 시간대별 메인 케어 + 자유 케어.
- **5대 액티비티**: 깨우기(기상)·아침/점심/저녁밥·재우기 — 각 1회/일, 욕구 회복+친밀도+일기 조각 노출
  - ✅🔒 **6/11 L24 재확정**: 메인 5종 = **깨우기·아침·점심·저녁·재우기** (사이트맵 Activity_Main_01~05의 02~04 "미정"은 본 구성으로 확정. M10 일기 5조각 1:1 유지)
  - **스토리100 배분**(6/11 신설 레이어): 메인 슬롯별 대사/스토리 풀 100개를 배분 — 깨우기·재우기(잠자리 이야기) 포함, 상세는 M10과 연동 설계
- **자유 케어**: 쓰다듬기·말걸기·간식·옷갈아입히기·씻기기·산책·놀이·사진·선물 (쿨다운·1일 캡)
- **우연 케어**: 친구가 말 걸어옴("오늘 머리 어때?"·"같이 갈래?") — 들렀을 때 확률 발생
- **상태**: ✅

### (미세 시간 이벤트)
- 1시간 단위 친구 행동/자세 변화("거울 보고 있음"). 들를 때마다 새 모습 → 호기심 회귀.
- 이스터에그 시각(별꼬리 01:17 등).

## 3. 파라미터·욕구 모델 (둘 다 제시 — C-1 결정 대기)

**A안 — 다마고치 6욕구**: hunger·energy·social·hygiene·fun·health (0~10), 시퀀스당 -1, 밀림 허용, mood = 6욕구 평균.
**B안 — GDD 4파라미터**: Hunger·Clean·Mood·Energy (0~100), 시간 자연 감소, 미수행 패널티.

> ✅🔒 **확정(6/9)**: **4파라미터 표면(Hunger·Clean·Mood·Energy, 0~100) + 다마고치 시퀀스·밀림 로직**(접속 트리거·안 들러도 손해 X).

## 4. 친밀도·진화

| 모델 | 다마고치: Lv0~10(호칭 변화·외형 진화·액션 해금) / GDD: Lv1~30(팬 호칭 단계) |
| 미수행 | 친밀도 **음수 X**(헌법). 일기 톤만 "관심 못 받은 흔적" |
| 일일 캡 | 다마고치=캡 제거(+5~+15/일) / GDD=행동별 |

> 100일 = Lv10(다마고치) = 일기 100편(500조각) 1단위. ↔ GDD Lv30(반짝섬 전 구역 완성).

### 5-3. 고려사항
- 케어 = **숙소(M04) 구동** (메인맵=지도). 무대는 구역(M06/M08).
- 메인 5번 ↔ 일기 5조각(M10) 1:1 = 케어와 읽을거리 한 흐름
- 다마고치 "밀림 허용/접속 트리거" = **안 들러도 손해 X**(헌법) — 강력 채택 후보
