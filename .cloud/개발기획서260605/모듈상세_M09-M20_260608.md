## M09. 도감·수집

- **목적**: 발견 기록 + **진화 연료**(아이동 귀속 25 아이템).
- **대조**: GDD §10 7종 도감 / 매뉴얼 S03 도감 트랙 / 6/8 25아이템→진화.
- **화면**: CODEX — 7종 도감(포토카드·K-POP용어70·아이템 아카이브·탐험15·레시피·음악·주민) + 아이템 재주문(원가×1.2).
- **연계**: 배치 생산(M06)→귀속 아이템→진화(M08).
- **미결·권장**: 도감→진화 수치(G-5) ⭐ 종별 25 중 필요분만 소비(전부 X). / 첫 케어=도감 본등재(다마고치 6-0 정합) ✅.

## M10. AI 페르소나·꿈 일기 ★핵심

- **목적**: 유저 데이터 기반 **월간 진화** 대사 + 꿈 일기. 차별점.
- **대조**: GDD §15 / 다마고치 "메인 5번=일기 5조각" / Visual View(M15) 키워드.
- **화면**: TALK(말풍선 대화·선택지) · DREAM(재우기→꿈 에피소드) · LETTER(편지·AI 답장).
- **작동**: 유튜브 링크/해시태그 → 키워드 추출 → 월간 랭킹 → 말투·대사 테마. RLHF(별점 1~5).
- **연계**: 케어 메인5(M07)↔일기 5조각 / Visual View 1위 키워드→대사(M15).
- **미결·권장**:
  - 일기 양식: **5조각/일**(다마고치) vs **통편**(페르소나 skill) → ⭐ **둘 병존**: 일일 5조각 + 마일스톤 통편(Lv3·5·10·100일).
  - AI 안전: 🟠 ⭐ **입력(유튜브/해시태그) 모더레이션 + 출력 금칙어 필터**(실명·민감어). 어린 팬층 필수.
  - 비용: 🟠 월간 배치 생성(실시간 X)로 LLM 비용 절감.

## M11. 포토카드 ★핵심

- **목적**: 케어 기록 반영 **실시간 생성** 카드 = 세상 단 하나. 핵심 수집·공유.
- **대조**: GDD §11 6등급 / S02 데뷔 포토카드(html2canvas·S3·OG).
- **화면**: BINDER(My Binder·전시·공유) — 등급 일반60/레어25/슈퍼레어10/울트라4/전설1/한정.
- **생성 변수**: 성장단계·의상·AI 키워드·**4파라미터/친밀도 Lv30**·구역·계절 (✅ A-1·A-2 확정 반영).
- **활용**: SNS · UGC 장터 거래(M14) · 친구 선물(M16) · 마이룸 액자(M12).
- **미결·권장**: 생성 비용·이미지 파이프라인 ⭐ **합성형(레이어드 PNG + 등급 프레임)**, 생성형 X(비용·IP안전). / S02 "거래 X" ↔ GDD "장터 거래" 충돌 → ⭐ **중복 카드만 거래**(소장 원본 보호).
