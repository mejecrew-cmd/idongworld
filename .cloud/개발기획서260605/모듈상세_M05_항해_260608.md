# 🧩 M05. 항해 (Voyage) — 상세 명세

> **위상**: 게임 핵심 루프. 항구에서 출항 → 부루마블 보드 → 아이동 섬 만남 → **영입 + 마이섬 15구역 슬롯 편입**(M06).
> **상태**: 🟢 구현(PoC v3) · ✅ 확정 · 🟡 기획중 · 🟠 미결

---

## 0. 개요
- **흐름**: `항구 → 항로 선택 → 보드(30칸) → 칸 도착 → (아이동 섬 상륙) → 영입 시나리오 → ① 아이동 편입(숙소) ② 아이동 섬 편입(15구역 슬롯)`
- **6/8 핵심**: 보드에서 만난 **아이동 섬을 마이섬 고정 15구역의 빈 슬롯에 편입**. (M06 연계)

## 2. 화면별 상세 명세

### HARBOR · 항구
- **영역**: 마이섬 **AREA-02 감정의 파도 항구**(앵커, 6/11 L23) — 항해 출입구
- **목적**: 항로 선택·출항 + 아이템 매매 + 배/항구 업그레이드.
- **진입**: 메인맵 항구 구역 탭
- **UI 요소**: 사이드뷰 다이오라마 · 정박된 조각배(Lv) · 항구(Lv) · `[항로 선택]` · `[입항/짐 비우기]` · 매매
- **이탈→다음**: 항로 선택 → BOARD / 입항 → 적재 비우기
- **상태**: ✅ (PoC HARBOR-01)

### ROUTE · 항로 선택
- **영역**: 항구 내
- **목적**: 항로 = 보드 30칸 구성을 결정.
- **항로(Phase 1)**: **산책 항로**(기본). (나와바리=친구섬 Phase3 / 고양이귀·별빛·계절 Phase2~)
- **이탈→다음**: 선택 → BOARD(그 항로의 30칸)
- **상태**: 🟡 (산책만 Phase1)

### BOARD · 항로 보드 (30칸)
- **영역**: 바다 보드
- **목적**: 지정 이동을 기본으로 칸 도착 처리, 주사위 옵션으로 항해 리듬을 가속.
- **UI 요소**: 모바일 세로 30칸 루프(상5·좌우10·하5) · 30번=마이섬(입항) · 중앙 패널(현재 칸 이름·타입·설명·행동) · 주사위
- **이동**: ✅🔒 **지정 이동 기본 + 주사위(가속) 옵션** (6/9 확정. 첫 사용자 명확성↑)
- **규칙**: 모든 칸 통과 시 자재 1개 / 한 바퀴 완주 보너스
- **이탈→다음**: 섬 칸 → 상륙 / 마이섬 칸 → 입항
- **상태**: 🟢 (PoC SEA-01) / ✅ 이동 정책 확정

### TILE · 칸 도착
- **목적**: 슬롯 종류 판정 → 처리.
- **분기**: 빈바다(자재)·폭풍(손실)·보물(보상컷)·섬(상륙 버튼)
- **상태**: 🟡

### ISLE · 아이동 섬 상륙
- **목적**: 아이동 페르소나 시각화 공간(HERO of KINGDOM식) 탐험 → 영입 진입.
- **이탈→다음**: 영입 조건 충족 → RECRUIT
- **상태**: 🟡

### RECRUIT · 영입 시나리오 → 편입
- **목적**: 영입 1컷 후 **두 편입 실행**.
- **두 편입** (→ [아이동편입_아이동섬편입_절차초안_260608.md])
  - ① **아이동 편입**: 캐릭터 → 숙소(M04) 마당/방 합류
  - ② **아이동 섬 편입**: 그 섬 콘텐츠(테마·미니게임·자원) → **마이섬 15구역 빈 슬롯(M06)**
- **상태**: 🟡

## 3. 칸 분류 + 산책 항로 분배 (✅ 6/11 L25 재정의)

> **분류(L25 확정)**: **기본칸(빈바다) + 이벤트 타일 5종(자원섬·아이동섬·가챠·보물·폭풍) + 친구섬(Phase3 예약)**.
> 사이트맵의 "TILE1~5 타일 5종" = 이벤트 타일을 가리킴. 빈바다는 타일 종류가 아닌 기본 칸.

| 슬롯 | 분류 | 내용 | 산책 항로 30칸(가설) |
|---|---|---|---|
| 빈 바다 | 기본칸 | 자재 1개 자동 | 10 |
| 자원섬 | 이벤트 타일 | 자재 T1~T3 채집 | 6 |
| **아이동 섬** | 이벤트 타일 | 영입 → **15구역 편입** | 5 |
| 가챠 칸 | 이벤트 타일 | 영입 전=신규 / 후=시즌 손님 | 2 |
| 보물 | 이벤트 타일 | 일회성 자재·코인 | 4 |
| 폭풍 | 이벤트 타일 | 정체·자재 일부 손실 | 3 |
| 친구섬 | 예약(Phase3) | 다른 유저 하트섬(나와바리) | 0 (Phase3) |

### 5-2. 쟁점
| 쟁점 | A | B | 메모 |
|---|---|---|---|
| 이동 | **항해 맛·PoC 구현** | **첫 사용자 난이도**(6/5 반론) | ✅ 지정 이동 기본 + 주사위 옵션으로 해결 |
| 아이동 섬 위상 | **보드 슬롯**(만나는 곳) | **마이섬 구역**(편입 후) | 둘 다 — 만남=보드, 거주=구역 편입 |

### 5-3. 고려사항
- **배 적재량**(6/5) = 신규 레이어 → 자재 인벤토리 한계·입항 비우기·배 업그레이드
- 아이동 섬 = **페르소나 시각화 공간**(HERO of KINGDOM식) → 편입되면 그대로 마이섬 구역 콘텐츠
- 가챠 칸 = 본진 영입 전(신규)/후(시즌 손님 3일) **의미 전환**
- 첫 만남(어푸어푸)은 보드 밖 = 오프닝(M02). 이후 만남은 보드.
- 친구섬/나와바리 = 소셜(M16), Phase 3
- 안티골: 헤비 그라인드·항로 과다·폭풍이 잔잔함 깨기 금지
