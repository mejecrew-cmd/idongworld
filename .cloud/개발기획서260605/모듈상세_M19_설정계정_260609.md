# 🧩 M19. 설정·계정·시스템 — 상세 명세

## 0. 개요
- 앱 설정·계정(연동/전환/탈퇴)·통계·CS·다국어. 데스크톱 확장 UI 후보 보관.

## 1. 기획 대조
- GDD §23 / UI v2 L0(설정·유저정보) / 6/8(계정관리 M01→M19 이관) → ✅ 통합.

## 2. 화면별 상세
- **APP-SET**: 사운드·BGM·볼륨·알림 토글.
- **ACCOUNT**: 소셜 연동·계정 전환·탈퇴.
- **STATS / CS / I18N**: 통계·고객지원·다국어.

## 3. 서브모듈 · 예시값 (불렛)
- **앱 설정**: 사운드·BGM·볼륨 · 알림 on/off · 푸시 설정
- **계정**: 소셜 연동(구글·애플·X·카카오·네이버·LINE) · 계정 전환 · **탈퇴**(M01에서 이관)
- **통계·활동 기록**: 플레이 기록·누적 출석·도감 진행
- **고객지원(CS)**: 문의·공지·FAQ
- **다국어(I18N)**: 70종 키 구조 / ⭐ **Phase1 = 한·영**, 키만 70 대비
- **데스크톱 확장 UI(후보)**: 퀘스트 상세 · 편지 전문 · 미니맵/범례 · 디버그 패널(개발)

## 4. 미결 · 쟁점 · 고려
- 미결: 다국어 적용 범위(Phase1 한영) · 데스크톱 확장 UI 우선순위(후순위)
