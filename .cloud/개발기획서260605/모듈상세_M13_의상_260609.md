- 부위별 코디 → 비주얼 숙련도·포토카드 의상 스냅(M11). 성장 단계마다 착용 풀 확장.

## 1. 기획 대조

  - 진입: 숙소(아이동 방) · 이탈: 데뷔 의상(M08)·포토카드(M11)

## 3. 서브모듈 · 예시값 (불렛)
- **의류 카테고리(10)**: 상의 · 하의 · 원피스 · 신발 · 양말 · 모자 · 안경 · 가방 · 목걸이 · 풀코디 세트
- **성장 단계 연동**: 연습생=트레이닝·교복·기본 / 데뷔조=데뷔의상·MV컨셉 / 활동=무대·콘서트·팬미팅 / 탑=전설(전용 최고 등급)
- **패션 챌린지**: 주 1회 · 테마 코디(예: "여름 해변"·"콘서트 현장") · 커뮤니티 투표 → 한정 아이템
- **전용 의상**: 외부 IP 콜라보 시점만(6/5)

## 4. 미결 · 쟁점 · 고려
- 미결: 의상 슬롯 수 → ⭐ **Phase1 3~4슬롯**(작화 부담), 8슬롯은 후속
- 쟁점: 전용 의상 vs 어울림 속성(6/5) — 콜라보 전엔 어울림 속성만
- 고려: 레이어드 PNG 기반(베니월드 방식) · 종별 기본 포즈 선행 필요
