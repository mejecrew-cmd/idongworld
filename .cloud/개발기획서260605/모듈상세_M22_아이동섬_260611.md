# 🧩 M22. 아이동섬 — 상륙·탐험·영입 (스텁)

> **위상**: 항해(M05) 보드에서 아이동섬 칸 도착(ISLE) 후 진입하는 **상륙 전체화면(마이섬형 확장)**. 아이동 페르소나 시각화 공간 탐험 → 섬 퀘스트 → 영입 확정 처리(RECRUIT, M06으로 인계).
> **상태**: 🟡 기획중 (스텁 — 화면별 상세화 필요)

## 0. 개요
- 동선: `BOARD(M05) → ISLE(칸 도착) → iDONGisland_000(상륙 메인) → 섬 퀘스트/탐험 → RECRUIT(영입 시나리오, M06) → ① 아이동 편입(숙소) ② 아이동섬 편입(15구역 슬롯)`
- 위상 구분: **만남 = 보드(M05) / 상륙·탐험 = 아이동섬(M22) / 편입 규칙 = M06**. 편입 후 섬 콘텐츠(테마·미니게임 스킨·생산)는 마이섬 구역 레이어로 전환(L09/Z-1 혼합 테마).

## 2. 핵심 규칙
- 아이동섬 = **페르소나 시각화 공간**(HERO of KINGDOM식, 5월 발견섬 6구역 자산 재사용).
- 영입 확정 처리는 섬 서브 화면에서 정리 → 이후 슬롯 편입은 M06 LOCK 규칙(P-2~P-6) 그대로.
- 첫 만남(어푸어푸)은 보드 밖 = 오프닝(M02). M22는 두 번째 만남부터.
