# 🧩 M14. UGC 공방·장터 — 상세 명세

> **위상**: 유저 제작(의상·가구·프레임) + 거래. 크리에이터 생태계. **Phase 2+ 권장**.
> 🟢 · ✅ · 🟡 · 🟠

## 0. 개요
- 유저가 직접 제작·등록 → 장터에서 코인 거래. 구매·착용 시 SNS 자동 공유 → 크리에이터 자연 홍보.

## 1. 기획 대조
- GDD §19(제작·등록·장터) / 6/5(전용 아이템·모듈화는 콜라보 시점) → ✅ 도입 시점 Phase 2+.

## 2. 화면별 상세
- **UGC-MAKE(공방)**: 제작 종류·바디 선택 → 업로드/편집 → 미리보기 → 운영 검수 → 발행.
- **UGC-SHOP(장터)**: 코인 거래 · 한정 수량 · 주간 인기 크리에이터 랭킹.

## 3. 서브모듈 · 예시값 (불렛)
- **제작 종류(4)**: 의상(6 바디 기준 2D 업로드) · 가구 소품(커스텀 그래픽) · 포토카드 프레임 · 마이룸 테마 세트
- **등록 프로세스**: 진입(친밀도 Lv15 + 50코인) → 업로드 → 미리보기 → 검수 → 발행
- **장터 규칙**: 코인 거래 · 크리에이터 80%(운영 20%) · 하트 50→10세트 확장 / 하트 200→100세트(운영 승인)
- **거래 대상**: UGC 의상·가구·프레임 + 포토카드 중복(M11)

## 4. 미결 · 쟁점 · 고려
- 미결: 도입 시점 → ⭐ **Phase 2+**(Phase1 제외) · 운영 검수 처리량
- 쟁점: 개방형 UGC ↔ IP/저작권/금칙어 리스크 → 검수 게이트·필터 필수
- 고려: 생성형 업로드 IP·실명 필터(M10·M15와 공통) · 미성년 결제 보호

## 5. 다음
- Phase 2 진입 시 검수 SOP·수수료·저작권 정책 본설계.
