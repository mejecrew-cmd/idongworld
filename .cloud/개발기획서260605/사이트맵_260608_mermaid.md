## C. 유저 흐름 (Mermaid · 보조)

> 인벤토리(A)가 "무엇이 있나", 흐름(C)이 "어떻게 이동하나". 발표·동선 검토용 보조.

### C-1. 온보딩 (1회성 · 단방향)
```mermaid
flowchart TD
    login["로그인"] --> title["타이틀"] --> opening["오프닝 1~4"]
    opening --> meet1["첫 만남(어푸어푸 1)"] --> part2["상륙·폐허"]
    part2 --> gray["회색섬"] --> clean["청소 의식"] --> naming["이름 짓기"]
    naming --> hub["🏝️ 마이섬 허브"]
    classDef once fill:#ffeaea,stroke:#e57373,color:#000;
    class login,title,opening,meet1,part2,gray,clean,naming once
```

### C-2. 본 게임 루프 (허브 중심)
```mermaid
flowchart TD
    hub["🏝️ 마이섬 허브"]
    hub --> lodge["🏠 숙소"]
    hub --> harbor["⚓ 항구"]
    hub --> zones["🗂️ 고정 15구역<br/>앵커 2 + 편입 슬롯 13"]
    hub --> codex["📖 도감"]
    harbor --> voyage["⛵ 항해(부루마블)"]

    voyage -.자원·도감.-> codex
```

### C-3. 항해 → 편입
```mermaid

    harbor["⚓ 항구"] --> board["🎲 항로 보드"] --> tile{"칸 도착"}
```

### ▶ M01. 로그인 모듈 (Auth / 계정 진입)

**목적**: 게임 진입 전 인증·동의. **소셜 로그인 전용**(ID/비밀번호 없음).

**화면 구성**
| ID | 화면 | 내용 | 상태 |
|---|---|---|---|
| AUTH-SPLASH | 스플래시 | 로고·버전 체크·**자동 로그인** 토큰 확인 | ✅ |
| AUTH-LOGIN | 로그인 | 소셜 버튼 N종 + 자동 로그인 토글 | 🟢 |
| AUTH-TERMS | 약관 동의 | 필수(이용약관·개인정보) + **연령 정보 체크** + 선택(마케팅) | 🟡 |
| AUTH-TOS | 이용약관 전문 | 약관 본문 페이지 | 🟡 |
| AUTH-PRIVACY | 개인정보 처리방침 전문 | 본문 페이지 | 🟡 |
| AUTH-LOGOUT | 로그아웃 | 진입 전 안전 이탈·계정 전환 | 🟡 |

**규칙·결정**
